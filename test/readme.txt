How to test K_fit.py script:

On your platform you should have Python3 installed and following libraries:
Numpy
Pandas
Scipy

The present directory include all needed data and files to conduct calculations.
They are in dmph directory.

Run below statment in linux terminal.
python3 K_fit.py -20 20 < paths2files.txt > coefficients2.out

Your results should be the same included in coefficients.out text file.

More information about input data you could read in help.pdf file.
